Developer Documentation

Build Scripts:

Please Make Sure to run “npm i”

in the API folder to download all

dependencies

Go to API folder with command prompt

npm run start-server

  • Hosts webpage on localhost:8080

npm run ci

  • Starts server and starts tests

npm run test

  • Starts only the test

Schemas:

App Schema

(_id = App Name)

{

    _id: String,

    description: String,

    organization: String,

    platforms: [String],

    versions: String,

    link: String,

    price: Number

}

Form Schema

(_id = User name of Forms owner)

{

    _id: String,

    nameApp: String,

    description: String,

    organization: String,

    platforms: [String],

    versions: String,

    link: String,

    price: Number

}

Account Schema

(_id = Username)

{

    _id: String,

    password: String,

    userType: String

}

API Endpoints:

Account Endpoints

/Accounts/Users - GET

  • Returns all Accounts inside the database.

/Accounts/getUser/:id - GET

  • Id = Username
  • Returns Account Information

/Accounts/newUser - POST

  • Request Body must be filled and follow schema
  • Adds an Account to the database

/Accounts/UpdateUser/:id - PUT

  • Id = Username
  • Request Body must be a JSON
  • Updates Users

/Accounts/deleteAccount/:id - DELETE

  • id - Username
  • Removes Account

App Endpoints

/Apps/getApps - GET

  • Returns all Apps inside the database.

/Apps/getApp/:id - GET

  • Id = App name
  • Returns App Information

/Apps/sendApps - POST

  • Request Body must be filled and follow schema
  • Adds an App to the database

/Apps/sendApps/:id - PUT

  • Id = App name
  • Request Body must be a JSON
  • Updates App

/Apps/deleteApp/:id - DELETE

  • id - App Name
  • Removes App

Form Endpoints

/Apps/getForms - GET

  • Returns all Forms inside the database.

/Apps/getForm/:id - GET

  • Id =  Username
  • Returns Form Information

/Apps/sendForms - POST

  • Request Body must be filled and follow schema
  • Adds a Form to the database

/Apps/sendForm/:id - PUT

  • Id = Username
  • Request Body must be a JSON
  • Updates Form

/Apps/deleteForm/:id - DELETE

  • id - Username
  • Removes Form

Endpoints

/UserSession/:id

  • This endpoint gives back a session id for the user
